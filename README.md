# Introdução ao React

Este é um projeto de aprendizado de React. O objetivo é explorar os conceitos básicos e avançados do React, criando componentes reutilizáveis e compreendendo o ciclo de vida dos componentes.

## Objetivos

- Entender a estrutura de um projeto React
- Criar componentes funcionais e de classe
- Gerenciar estado e propriedades
- Utilizar hooks
- Integrar com APIs externas

## Como começar

1. Clone este repositório:
  ```bash
  git clone https://github.com/VictorTerres/introducao-react
  ```
2. Instale as dependências:
  ```bash
  npm install
  ```
3. Inicie o servidor de desenvolvimento:
  ```bash
  npm start
  ```
